Завдання: Згенерувати імітовставку (Message Authentication Code, MAC) для наданого шифротексту, створеного шифром AES в режимі CBC.
Для реалізації скористаємося бібліотекою pycryptodome.

Для генерації імітовставки використаю алгоритм HMAC-SHA256, який є поширений та безпечний. Для вхідних даних створимо окремий ключ на основі ключа використаний для зашифрування, оскільки використовувати один ключ для шифрування та автентифікації не рекомендується з міркувань безпеки. Для створення ключа використаємо функцію HKDF.
Після цього використаємо функцію HMAC-SHA256 на основі ключа автентифікації, шифротексту і вектора ініціалізації для створення імітовставки.
